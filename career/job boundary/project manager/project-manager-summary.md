# Project Manager

## 1. Core Responsibility

Project Manager owns:

-   Timeline
-   Resource coordination
-   Task tracking
-   Risk management
-   Delivery control

Project Manager ensures execution happens on time.

------------------------------------------------------------------------

## 2. What Project Manager Must Master

### 2.1 Planning & Scheduling

-   Milestone planning
-   Sprint planning
-   Dependency tracking

### 2.2 Risk Management

-   Identify blockers
-   Escalate issues
-   Adjust timeline when needed

### 2.3 Resource Allocation

-   Assign ownership
-   Balance workload
-   Coordinate cross-team work

### 2.4 Progress Monitoring

-   Status reporting
-   Burndown tracking
-   Delivery transparency

### 2.5 Communication & Alignment

-   Organize meetings
-   Remove friction
-   Ensure everyone knows priorities

------------------------------------------------------------------------

## 3. What Project Manager Does NOT Own

-   Business strategy
-   Feature definition
-   Technical design
-   Implementation details

Project Manager manages execution, not product direction.

------------------------------------------------------------------------

## 4. Boundary Summary

Product Manager:

> WHAT & WHY

Engineering:

> HOW

Project Manager:

> WHEN & WHO
